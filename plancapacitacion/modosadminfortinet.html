Equipos y Configuración:

Para este curso vamos a utilizar un Firewall un dispositivo FortiGate 60E lanzado en 2016 como parte de la serie FortiGate E.

Este dispositivo cuenta con un sistema operativo FortiOS v6.2. Cuenta con 7 puertos ethernet, un puerto DMZ "Zona Desmilitarizada" (el cual nos permite que un servidor pueda ser accesible desde internet).

Me gustaria hablar de los modos de acceso y de configuracion con los que cuenta FortiGate, ya que el video anterior, y este segundo video no se encuentran en el curso, pero para mi parecer son importantes ambas introducciones asi que se crearon.

📌 Modos de Administración en FortiGate

1️⃣ Modos de acceso a FortiGate

Método Descripción Comando para acceder
Interfaz Web (GUI) Acceso gráfico vía navegador (https://IP_FortiGate) No aplica
CLI vía SSH o Consola Administración mediante comandos ssh admin@IP_FortiGate o acceso por consola
Acceso por API Para automatización y gestión remota Requiere configuración previa

2️⃣ Modos de Configuración en CLI

Modo Descripción Prompt en CLI
Modo Usuario Acceso básico con comandos de monitoreo (get, execute) FortiGate-VM64 $
Modo Privilegiado Permite entrar a configuración con config FortiGate-VM64 #
Modo Configuración Permite cambiar ajustes dentro de secciones específicas (config system interface) FortiGate-VM64 (interface) #

📌 Configuración de Interfaces de Red

config system interface
edit port1
set ip 192.168.1.99 255.255.255.0
set allowaccess ping https ssh
next
end

📌 Configuración de Políticas de Firewall

config firewall policy
edit 1
set srcintf "port1"
set dstintf "port2"
set srcaddr "all"
set dstaddr "all"
set action accept
set schedule "always"
set service "ALL"
set logtraffic all
next
end

📌 Configuración de Rutas Estáticas

config router static
edit 1
set dst 0.0.0.0 0.0.0.0
set gateway 192.168.1.1
set device "port1"
next
end

📌Configuración de Usuarios y Autenticación

config user local
edit "usuario1"

end
