🔐 Introducción al uso de VeraCrypt para cifrar directorios y unidades extraíbles

En este video aprenderemos a utilizar una herramienta muy útil para cifrar directorios. Además, nos será de gran ayuda si queremos cifrar unidades de almacenamiento extraíble como discos duros, memorias USB o pendrives.

¿Por qué usar esta herramienta?

Porque es multiplataforma y bastante conocida. Esto significa que, si queremos transportar información entre diferentes sistemas operativos, podemos contar con su compatibilidad en Windows, macOS, Linux y otras plataformas reconocidas.

Aunque no está disponible directamente en los repositorios, desde su página oficial podemos descargar un paquete .deb compatible con todas las distribuciones basadas en Debian. La instalación es bastante sencilla.

También existe la posibilidad de descargar el código fuente y compilarlo según nuestro sistema, si así lo preferimos.

Descarga e instalación de VeraCrypt

La herramienta se llama VeraCrypt. Podemos buscar "VeraCrypt download" en Google para ir directamente a su sitio oficial (veracrypt.fr) y acceder a la sección de descargas.

Allí veremos que está disponible para los principales sistemas operativos. En mi caso, como uso Debian 10, descargaré la versión para consola. También está disponible con interfaz gráfica, aunque esta puede requerir más recursos y espacio.

Una vez descargado el paquete .deb, lo instalamos desde la terminal utilizando el comando:

sudo dpkg -i nombre-del-paquete.deb

Crear un volumen cifrado

Para cifrar un directorio, no necesitamos ser usuario root. Esto es una gran ventaja, ya que cualquier usuario puede proteger su información sin necesidad de permisos administrativos.

Ejecutamos el siguiente comando para crear un volumen:

veracrypt -c

Este comando inicia un asistente que nos guiará paso a paso para crear un archivo cifrado. Este archivo actuará como un contenedor o volumen. Cuando lo montamos, podemos acceder a la información; cuando está desmontado, queda completamente cifrado.

Parámetros importantes durante la creación:

  • Ubicación del volumen: por ejemplo /home/alumno/cifrado. Es recomendable que el nombre no revele información.
  • Tamaño: en este ejemplo usaremos 5 GB.
  • Algoritmo de cifrado: AES suele ser suficiente.
  • Sistema de archivos: si planeamos usarlo solo en Linux, podemos usar ext4. Si se usará en otros sistemas, se recomienda FAT o exFAT (aunque con limitaciones en archivos grandes o permisos).
  • Contraseña: debe ser fuerte; el programa nos advertirá si es demasiado débil.
  • Keyfile (opcional): podemos omitirlo si no queremos utilizar claves externas.
  • Generación de entropía: deberemos teclear aleatoriamente durante unos segundos para generar una clave robusta.

Una vez completado el proceso, se creará el archivo cifrado en la ubicación que elegimos.

Montar el volumen

Para montar el volumen cifrado y acceder a su contenido:

1. Creamos un directorio, por ejemplo:code mkdir ~/mis_datos

2. Montamos el volumen: veracrypt /home/alumno/cifrado ~/mis_datos

Ingresamos la contraseña. Si no usamos PIN ni keyfile, simplemente presionamos Enter cuando se nos pregunte por ellos. Una vez montado, el contenido del volumen se podrá gestionar como cualquier directorio normal. Cualquier archivo que guardemos dentro quedará automáticamente cifrado.

Desmontar el volumen

Cuando queramos ocultar nuevamente el contenido:

veracrypt -d ~/mis_datos

Esto desmontará el volumen, y el directorio quedará vacío. Para volver a acceder, simplemente repetimos el montaje como se explicó antes.
