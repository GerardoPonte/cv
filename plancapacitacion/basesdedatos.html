Plan de Capacitación: SQL Server

Este es el plan de aprendizaje permanente propuesto para fortalecer mis habilidades en SQL Server y brindar mayor soporte en la admnistracion de sistemas. Aquí detallo los temas propuestos a aprender y los proyectos basicos que algunos se han realizados:

1. Conceptos Básicos

  • Introducción al motor de base de datos SQL Server.
  • Creación y gestión de bases de datos:
    • Comandos CREATE DATABASE y configuración de rutas para archivos .mdf y .ldf.
  • Tipos de datos y diseño de tablas:
    • Tipos: INT, VARCHAR, DATETIME, etc.
    • Creación de tablas con CREATE TABLE.

2. Consultas Básicas

  • Uso de SELECT para recuperar datos.
  • Filtros con WHERE para condiciones específicas.
  • Ordenamiento con ORDER BY.
  • Consultas con funciones de agregación: SUM, AVG, COUNT, etc.

3. Modificación de Datos

  • Inserción de datos con INSERT INTO.
  • Actualización de datos con UPDATE.
  • Eliminación de datos con DELETE.

4. Relaciones entre Tablas

  • Claves primarias (PRIMARY KEY) y foráneas (FOREIGN KEY).
  • Uso de JOIN (Inner Join, Left Join, Right Join, Full Join) para combinar tablas.

5. Prácticas Avanzadas

  • Creación de índices para optimizar el rendimiento.
  • Automatización con triggers.
  • Vistas (CREATE VIEW) para simplificar consultas complejas.
  • Organización de bases de datos con esquemas.
  • Configuración de permisos para tablas y vistas.
  • Respaldo y restauración de bases de datos.

7. Ejercicios y Proyectos Realizados

  • Ejercicios prácticos con la base de datos AdventureWorks2014:
    • Consultas avanzadas.
    • Comparaciones entre vistas y tablas.
  • Proyectos:
    • Productos: Tablas relacionadas como Proveedores, Inventario y Ventas.
    • Escuela: Relaciones entre alumnos, materias y profesores.

8. Objetivo Final

Dominar T-SQL para consultas y administración avanzada, implementando proyectos completos que incluyan diseño, desarrollo y optimización de bases de datos.
