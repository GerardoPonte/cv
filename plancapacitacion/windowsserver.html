Plan de Trabajo: Administración de Windows Server y Active Directory

Introducción a Windows Server

  1. Instalación y configuración inicial de Windows Server.
  2. Configuración de roles y características (e.g., Active Directory, DNS, DHCP).
  3. Administración básica de usuarios y permisos.

Gestión de Active Directory

  1. Creación y administración de dominios.
  2. Alta, baja y modificación de usuarios y grupos.
  3. Implementación de políticas de contraseñas y bloqueo de cuentas.
  4. Configuración de relaciones de confianza entre dominios.

Gestión de GPOs (Group Policy Objects)

  1. Creación y aplicación de GPOs para configuraciones de seguridad.
  2. Restricción del acceso a ciertas aplicaciones o servicios.
  3. Configuración de scripts de inicio y cierre de sesión.
  4. Monitoreo y solución de problemas relacionados con GPOs.

Seguridad en Windows Server

  1. Configuración de firewalls y reglas de seguridad.
  2. Auditoría de inicio de sesión y acceso a recursos.
  3. Gestión de certificados digitales para seguridad.
  4. Implementación de actualizaciones y parches críticos.

Administración de Recursos

  1. Configuración de carpetas compartidas y permisos NTFS.
  2. Implementación de cuotas de disco para usuarios y grupos.
  3. Gestión de impresoras en red y asignación por GPO.

Mantenimiento Preventivo y Correctivo
