Práctica, Accediendo a Fortinet y conociendo su entorno

En esta primera clase se realiza la configuración del entorno de laboratorio de Fortinet mediante la instalación de GNS3, VMWare, entre una serie de archivos de configuración que nos van a permitir la configuración del entorno de trabajo.

Por el momento abortamos la instalación y configuración del laboratorio en el entorno de ParrotOS, por temas de configuración, y optamos por continuar bajo Windows 11.

1.- Primero, tenemos que realizar la instalación y la configuración de GNS3VM.ova.

2.- Vamos a omitir algunos pasos para no saturar de imágenes el sitio, para mi es importante realizar estas prácticas ya que, para un servidor, son como si hicieras planas, se me graban mejor las cosas.

Ya una vez configurado el laboratorio vamos a realizar el primer ingreso al primer Fortinet del Site-1 dando clic derecho al mismo y seleccionando (console) y con el comando (get system interface) obtenemos la dirección IP para ingresar a nuestro Fortinet.

3.- Posteriormente dentro de la consola del Fortigate tenemos una serie de opciones tanto lateralmente como en el centro. Se da una explicación de cada una de ellas para tener un mejor panorama de Fortinet 6. En la parte inferior se cuenta con una lista que describe el panel lateral izquierdo y tenemos una descripción de la pestaña u opción (Status)

Fortinet - Panel Lateral izquierdo

Dashboard

Resumen general del estado del firewall, incluyendo tráfico en tiempo real y estadísticas.

Security Fabric

Gestión centralizada de seguridad con dispositivos Fortinet conectados.

FortiView

Análisis visual del tráfico de red y amenazas detectadas.

Network

Configuraciones de red, interfaces, rutas y VLANs.

System

Administración del sistema, actualizaciones, configuración de administradores.

Policy & Objects

Definición de políticas de seguridad y objetos de red.

Security Profiles

Perfiles de seguridad como IPS, antivirus, filtrado web y control de aplicaciones.

VPN

Configuración y monitoreo de túneles VPN.

User & Switch Controller

Gestión de usuarios, autenticación y control de switches.

Log & Monitor

Registro y monitoreo de eventos y actividades del firewall.

Descricion de la pestaña Status:

System Informatión

Muestra detalles clave sobre la configuración y el estado del dispositivo Fortinet.

Licences

Información sobre las licencias activas y expiradas del sistema.

Virtual Machine

Estado y detalles de la virtualización del dispositivo, si aplica.

Fortigate Cloud

Acceso y configuración del servicio en la nube de Fortinet.

Security Fabric

Integración de dispositivos Fortinet para una seguridad unificada.

Security Rating

Análisis y puntuación del nivel de seguridad del sistema.

Administrators

Lista y gestión de usuarios con permisos de administrador.

CPU

Uso actual del procesador del sistema.

Memory

Consumo de memoria del dispositivo.

Sessions

Número de sesiones activas en el firewall.
